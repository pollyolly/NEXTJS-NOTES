# NEXTJS-NOTES
### Note
```
- by default all components in NextJs are Server Side.
  To allow client side insert directive in a component add "use client"; above the file.
```
Sample: Hero.tsx
```javascript
"use client";
import React from "react";
```
### Installation
```vim
$cd next-commerce
$npx create-next-app@latest ./
```
